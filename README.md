# Ratat-Attack

**Game Description**

This project is a local multiplayer game where two players battle for control of the kitchen. One player takes on the role of a VR Chef, while the other controls a Rat using a mouse and keyboard.

The Chef is responsible for preparing and serving burgers by assembling ingredients and completing orders as quickly as possible. At the same time, the Rat actively interferes by disrupting ingredients, blocking actions, and creating chaos in the kitchen.

The gameplay focuses on fast-paced interaction and competition, with each role offering a different play style, one centered on coordination and speed, and the other on disruption and strategy.
